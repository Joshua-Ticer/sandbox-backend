# Sandbox Backend

A sample backend repository with a simple CI/CD pipeline for demo and learning purposes.

## Overview

This repo contains a starter backend service designed to help developers learn how to build, test, and deploy a small API project with automated pipeline support.

## Features

- Simple backend API structure
- Local development setup
- Automated tests
- CI/CD pipeline integration for build and deployment

## Repository Structure

- `app/` - backend source code
- `test/` - automated tests
- `Dockerfile` - container image definition
- `.github/workflows/actions.yml` - CI/CD workflow definition


## CI/CD Pipeline

The CI/CD pipeline is configured to:

- install dependencies
- run linting and tests
- build the application or container image
- deploy to self-hosted environment on my laptop

This repo is suitable for learning how to connect a backend project to a pipeline service like GitHub Actions, GitLab CI, or another CI/CD provider.

## Learning Goals

- Understand backend project organization
- Practice writing automated tests
- Learn how CI/CD pipelines validate and deploy code
- Explore containerization and deployment workflows

## Notes

This repository is intended for demonstration and educational use, not for production.
